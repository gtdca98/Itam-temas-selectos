Notas CARGANEO4J.sh

CARGANEO4J.sh realiza vuelca el contenido  de un archivo 
con comandos en CYPHER en la interfase neo4j-shell

Es muy importante detener el motor neo4j antes de integrar información
vía neo4j-shell.

Comandos relacionados…

neo4j status ## muestra el estado del motor neo4j
neo4j stop   ## detiene el motor neo4j
neo4j start  ## inicia el motor neo4j
